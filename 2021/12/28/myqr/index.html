使用MyQR创建二维码

Posted on 2021-12-28 Edited on 2022-01-03

准备工作

MyQR是python的一个第三方库，用法比较简单，可以直接在cmd中使用。其他的库比如qrcode我没用过，暂时不清楚。

首先安装python环境并设置好环境变量，然后打开cmd，输入pip install MyQR，安装成功后输入myqr会显示如下画面：

直接在cmd中使用(推荐)

非常奇怪的是，在经历了一次重装系统并重新安装MyQR后，cmd中已经无法识别MyQR指令了。在前一个系统中我甚至没有为它添加过环境变量，而且在我更改过的清华镜像中无法通过pip直接安装MyQR也就排除了pip自动配置的可能性。从pypi下载的tar包安装后仍能正常使用，但cmd中已经失效。

所有的创建工作均在一行指令中完成。打开cmd，输入myqr "想展示的文字内容"或myqr 目标网址就可以生成一个默认状态的二维码。在后面可以附带各种参数，各参数的作用如下：

  • -v：生成的二维码图片的边长
  • -l：纠错等级、可识别度，即遮挡一部分后仍可被识别
  • -p：将二维码与指定图片或动图结合
  • -c：布尔值，为true时生成的二维码图片是彩色的
  • -con：对比度，默认1.0
  • -bri：亮度
  • -n：生成的二维码图片的文件名，默认为qrcode.png
  • -d：生成的图片的位置，默认为cmd的当前位置
